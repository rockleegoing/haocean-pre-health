---

### 2.2 激活页
**原型图**: `激活.png`

**功能点**:
- 页面标题：设备激活
- 输入项：激活码（6-8 位字母数字组合）
- 错误提示：无效激活码（红色提示）
- 提交：确认按钮

**业务流程**:
```
首次启动 App → 检查本地激活状态 → 未激活跳转此页
→ 输入激活码 → 后端验证 → 激活成功 → 设置登录密码
```

**开发参考**:
- 新增页面 `pages/activate/index.vue`
- 后端接口：`POST /api/device/activate`

## 三、快速出证模块（首页）

### 3.1 首页
**原型图**: `首页.png`

**功能点**:
- 顶部单位选择器：显示当前选中单位（下拉选择）
- 便捷执法区（5 个功能）：
  | 功能 | 图标 | 原型图 |
  |------|------|--------|
  | 快速拍照 | 图片图标 | `便捷执法_快速拍照.png` |
  | 快速录音 | 麦克风图标 | `便捷执法_快速录音.png` |
  | 快速录像 | 摄像机图标 | `便捷执法_快速录像.png` |
  | 快速导航 | 定位图标 | `便捷执法_快速导航.png` |
  | 执法记录 | 文件夹图标 | `便捷执法_执法记录.png` |
- 快速制作文书（6 种常用文书）：
  - 现场笔录、监督意见书、询问笔录
  - 现场照片、证据先行决定书、采样记录
- 其他文书模板（列表展示）：
  - 电子送达确认书、授权委托书
  - 接收调查通知书、相对人提供材料通知书

**数据结构**:
```javascript
{
  currentUnit: { id, name, industry },
  quickActions: [...],
  documentTemplates: {
    quick: [...],  // 快速制作文书
    other: [...]   // 其他文书模板
  }
}
```

**开发参考**:
- 修改现有 `pages/index/index.vue`
- 保留统计图表，增加文书模板入口

### 3.2 快速拍照
**原型图**: `便捷执法_快速拍照.png`

**功能点**:
- 调用设备原相机
- 拍照完成后自动保存至当前执法对象资料档案
- 支持连续拍照
- 照片自动关联：单位、行业、检查日期、执法人员

**技术实现**:
```javascript
uni.chooseImage({
  count: 9,
  sourceType: ['camera'],
  success: (res) => {
    // 保存到本地 + SQLite 记录
  }
})
```

**开发参考**:
- 新增页面或组件 `components/camera/QuickPhoto.vue`
- 后端接口：`POST /api/evidence/photo`

---

### 3.3 快速录音
**原型图**: `便捷执法_快速录音.png`

**功能点**:
- 调用设备录音机
- 显示录音时长（如 00:02）
- 显示声波动画

```javascript
recorder.start({ format: 'mp3' })
recorder.onTimeUpdate((res) => { /* 更新时长 */ })
```

- 后端接口：`POST /api/evidence/audio`

### 3.6 执法记录
**原型图**: 
- `便捷执法_执法记录.png`（列表页）
- `便捷执法_执法记录_导入图片.png`
- `便捷执法_执法记录_填写文书.png`
- `便捷执法_执法记录_快捷打印.png`
- `便捷执法_执法记录_快速上报.png`

**功能点 - 列表页**:
- 显示当前单位的所有执法记录
- 每条记录显示：照片数、录音数、录像数、文书数
- 操作：上报、编辑、查看、打印
- 状态标识：待上报、已上报、已归档

**功能点 - 导入图片**:
- 打开本地图库
- 多选上传图片
- 显示上传进度

**功能点 - 填写文书**:
- 选择文书类型（弹窗选择）
- 在线编辑文书内容
- 提交文书、打印文书

**功能点 - 快捷打印**:
- 选择已完成的文书
- 通过印刷打印机打印
- 打印设置：纸张大小、份数、打印机选择

**功能点 - 快速上报**:
- 选择待上报的记录
- 一键上报到后台
- 显示上报进度

**数据结构**:
```sql
enforcement_record:
  record_id, subject_id, subject_name, industry_id
  check_date, inspector_ids, status, is_sync
  photo_count, audio_count, video_count, doc_count
```

**开发参考**:
- 列表页 `pages/work/enforcement/list.vue`
- 详情页 `pages/work/enforcement/detail.vue`
- 后端接口：`GET /api/enforcement/records`

## 十、开发优先级

### P0（核心功能）
1. 登录/激活
2. 首页（单位选择、便捷执法入口）
3. 快速拍照/录音/录像
4. 执法记录（列表、详情、上报）
5. 添加单位/修改资料

### P1（重要功能）
6. 法律法规（列表、详情、搜索）
7. 规范用语（列表、详情）
8. 监管事项（列表、详情）
